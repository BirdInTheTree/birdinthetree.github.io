О проекте

tvbible — бесплатная база знаний по питчингу сериалов. Синтез шести книг о том, как писать питч-библию: Nash, Harris, Hauge, Landau, Oberg, Levy.

Каждая карточка — одна атомарная идея: принцип, ловушка, критерий или пример из шоу. Карточки связаны между собой через [[викиссылки]], образуя навигируемый граф знаний.

Сделала Nataliia. Бесплатно, без логина, без рекламы. Read in English.
